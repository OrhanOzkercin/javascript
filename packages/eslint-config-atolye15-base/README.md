# eslint-config-atolye15-base

This package provides AtÃ¶lye15's base JS .eslintrc (without React plugins) as an extensible shared config.

## Install

Install the correct versions of package.

Yarn

```bash
yarn add eslint-config-atolye15-base
```

Npm

```bash
npm install eslint-config-atolye15-base
```

## Usage

```
// .eslintrc

{
  "extends": [
    "eslint-config-atolye15-base"
  ]
}

```
